## 🌟 Overview

**Lofi Focus Player** is your ultimate companion for deep work, studying, or simply relaxing. It combines a curated lofi music experience with powerful productivity features like a Pomodoro timer, ambient noise mixing, and a real-time audio visualizer—all wrapped in a stunning, responsive glassmorphism UI.

## ✨ Features

- **🎵 Premium Lofi Player:** Seamlessly play, pause, and navigate through a relaxing playlist carefully designed for deep focus.
- **🎨 Glassmorphic Interface:** A beautiful, modern, and fully responsive semi-transparent design that looks gorgeous on any device.
- **⏱️ Focus Timer (Pomodoro):** Integrated customizable timer with predefined Focus, Short Break, and Long Break presets to boost productivity using the Pomodoro technique.
- **🌧️ Ambient Noise Mixer:** Overlay relaxing background sounds like Rain, Café murmur, or Forest ambiance. Mix and match to create your perfect study environment.
- **🌊 Real-time Audio Visualizer:** Dynamic, real-time waveform and frequency visualizers powered by HTML5 Canvas and the Web Audio API that respond to your music.

## 🚀 Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- Node.js (v16.x or later recommended)
- npm (Node Package Manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/lofi-focus-player.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd lofi-focus-player
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Open [http://localhost:5173](http://localhost:5173) to view the app in your browser!

## 🛠️ Built With

- **[React](https://reactjs.org/)** - For building the user interface.
- **[Tailwind CSS](https://tailwindcss.com/)** - For utility-first styling and achieving the gorgeous glassmorphism look.
- **[Vite](https://vitejs.dev/)** - For blazing fast build tooling and hot module replacement.
- **[Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API)** - For robust audio manipulation and real-time visualization.
- **[Framer Motion](https://www.framer.com/motion/)** *(Optional/Planned)* - For smooth micro-interactions and transitions.
